# Sales Data Analysis Pipeline using Azure Data Factory

## Project Overview

This project demonstrates data ingestion and movement using Azure Data Factory.

## Architecture

Sales CSV File
↓
Azure Blob Storage
↓
Azure Data Factory
↓
Processed Data Container
↓
Power BI

## Tools Used

- Azure Data Factory
- Azure Blob Storage
- GitHub
- Power BI

## Features

- Linked Services
- Datasets
- Copy Activity
- Monitoring
- Triggers

## Outcome

Automated movement of sales data for analytics reporting.
